How It Works

01

MediaPipe Hands

Your camera feed is analyzed frame-by-frame using Google's MediaPipe, detecting 21 hand landmarks in real-time.

02

Gesture Classification

Landmark coordinates are normalized and fed into a classifier that maps hand configurations to ASL signs.

03

Django REST API

Detected signs are POSTed to a Django backend, stored in SQLite, and available via REST endpoints for analytics.

04

Text & Speech Output

Recognized signs are displayed as text and synthesized to speech using the Web Speech API.
